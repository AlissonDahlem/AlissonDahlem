## 🧠 Engineering Principles

- **Idempotency > hope**
- **Tracing first** (if it’s not observable, it doesn’t exist)
- **Failure is a feature**: retries, DLQ, backpressure
- **Latency budgets** and cost-aware design
- **AI must be measurable** (evals, metrics, regressions)
